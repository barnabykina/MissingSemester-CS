## Topic 1: The Shell

Although there are several shells for various platforms, this course focuses on the bash terminal that is found on __Unix-based platforms__ such as Linux and MacOS. Windows users are highly encouraged to make use of either the __Windows Subsystem for Linux__ (commonly referred to as __WSL__) or a Linux __virtual machine__.

The shell is usually much faster than "clicking around", as the lecuturers put it, on a GUI. It also comes with expressive power that isn't easily found in any one graphical program.

> "...the shell gives you the ability to combine programs in creative ways to automate nearly any task.
> 
> Knowing your way around a shell is also very useful to navigate the world of open-source software (which often come with install instructions that require the shell), building continuous integration for your software projects (as described in the Code Quality lecture), and debugging errors when other programs fail."

_An excerpt from the [lesson's webpage](https://missing.csail.mit.edu/2026/course-shell/)._
